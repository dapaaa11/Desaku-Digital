Frontend (`cd frontend`):
- Start development server: `npm run dev`
- Build project: `npm run build`
- Start production server: `npm run start`
- Lint code: `npm run lint`

Backend (`cd backend`):
- Start development server: `npm run start:dev`
- Build project: `npm run build`
- Format code: `npm run format`
- Lint code: `npm run lint`
- Run unit tests: `npm run test`
- Run E2E tests: `npm run test:e2e`

Utilities:
Since this is a Windows environment, typical cmd/powershell commands are used (`dir`, `cd`, `Get-Content`, `Select-String`, etc.). However, Node.js scripts via `npm run` handle the main project workflows across platforms seamlessly.
