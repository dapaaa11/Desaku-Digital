DesaKu Digital is a web and mobile platform designed to improve village services through digitalization, information dissemination, and strengthening the local economy. It allows citizens to access information, request administrative services online, and use an MSME catalog for local products. Village officials can manage data, publish information, and monitor service requests through a centralized system.

Structure:
- `frontend/`: Next.js web application.
- `backend/`: NestJS backend server with Prisma ORM.
